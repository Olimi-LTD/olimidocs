---
title: 'Fetch Customers by Customer List'
openapi: 'GET /customers/{customer_list_id}'
---

Fetches all customers from the specified customer list by using the `customer_list_id`.

### Request Parameters

- `customer_list_id` (required): ID of the customer list to fetch the customers from.

### Response Example

#### Success (200 OK)

```json
[
  {
    "id": "abc123",
    "first_name": "string",
    "last_name": "string",
    "phone_number": "string",
    "email": "string",
    "gender": "string",
    "created_at": "2024-10-10T12:26:21.000000Z",
    "updated_at": "2024-10-10T12:26:21.000000Z"
  }
]
