---
title: 'Import Contacts'
openapi: 'POST /contacts/import'
---

Imports contacts in bulk using a CSV file and assigns them to the specified roster.

### Request Example

- **file**: The CSV file containing the contact information.
- **roster_id**: The ID of the roster to import the contacts into.

### Response Example

#### Success (200 OK)

```json
{
  "imported_count": 100
}
